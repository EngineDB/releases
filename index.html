Official APT repository for EngineDB packages for Ubuntu 24.04 LTS (Noble).

Ubuntu 24.04 LTS (Noble) - Installation

# Add GPG key
curl -fsSL https://enginedb.github.io/releases/enginedb.gpg | sudo gpg --dearmor -o /usr/share/keyrings/enginedb-archive-keyring.gpg

# Add repository
echo "deb [signed-by=/usr/share/keyrings/enginedb-archive-keyring.gpg] https://enginedb.github.io/releases/apt/ubuntu noble/enginedb/1.4" | sudo tee /etc/apt/sources.list.d/enginedb.list

# Update and install
sudo apt update
sudo apt install enginedb
